# Redux Task Manager

## Goals

- Implement a Redux store that manages a to-do list with async API calls (e.g., fetching tasks from a fake backend).
- _Stretch_: Add optimistic updates and error handling

## Scripts

- `dev`/`start` - start dev server and open browser
- `build` - build for production
- `preview` - locally preview production build
- `test` - launch test runner
